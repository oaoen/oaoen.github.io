泊-百度输入法-特点

百度输入法皮肤【泊】现已制作完毕，依然是由森大@森林里的小动物 设计、指导及提供代码#(墨镜滑稽) 主题为水。说实话真进去森大脑袋里看看是什么组成的，能源源不断有好创意出来，哈哈哈哈。

⛔️ 此皮肤基本涵盖了所有键位需求：【 九键、二十六键、左单手、右单手、十四键、十七键乱序、十八键 】快夸我！

工具栏依次对应菜单、全选、复制、剪切、粘贴、剪贴板、提示、联想。➠ 除手写外所有界面点击“提示”均可看到操作说明。

全局手势：

长按剪切清空文本 / 剪切左侧下划打开编辑 / 右侧下划打开表情 ▪️ 下划“水杯”隐藏面板 ▪️ 左/上划删除键清空输入码 ▪️ 长按中英切换键选择输入方式（快速切换手写） ▪️ 左/右划表情键打开搜索 ▪️ 上划空格切换中英 ▪️ 长按句号输入英文句号 ▪️ 上划回车打开符号 / 左划打开懒人短语 / 长按进入选字模式 ▪️ 上下左右移动光标、句首句尾等其余手势请查看提示页面

三.皮肤使用方法

💧 1005更新：【 14键、17键乱序、18键 】加入横屏模式；改善18键动效；修改空格背景样式；26键有输入码时数字键将变为分词键。

🌸 仙女们专属配色【 藕荷（粉） 】已上线#(欢呼) 修改了所有界面颜色！

🎨 新增12种配色：【 藕荷、勿忘草、蒂芙尼蓝、魅惑紫、青碧、深海绿、石绿、银鼠、辰砂、墨黑、酡红、🚥 】

⚠️ “水滴容器”按键丢失的问题请升级输入法至“官方最新版”解决 ／ 无法进入Emoji表情请更换官方皮肤选择Emoji后再启用本皮肤。

🛠 使用方法：点击文件使用百度输入法打开；或将文件移动到 存储/baidu/ime/skins ，之后从百度输入法超级皮肤中启用。
